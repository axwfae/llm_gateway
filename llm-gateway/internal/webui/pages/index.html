快速開始 Quick Start

  1. 在 服務器 Servers 頁面新增 LLM 服務器
  2. 在 模型 Models 頁面新增每個服務器支援的模型
  3. 在 API Keys 頁面新增 API Key
  4. 在 映射 Mapping 頁面設定本地模型名稱到服務器模型的映射
  5. 透過 API Port (:18869) 調用 /v1/chat/completions
